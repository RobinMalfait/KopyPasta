## KopyPasta

This is a plugin for various JetBrains products. It allows you to share your code, or partials of your code with anybody!

### Installation

1. Open PhpStorm
2. Preferences
3. Plugins
4. Search for `KopyPasta`, if you can't find it: ![](http://d.pr/i/1aWK1+)
5. Restart PhpStorm

### Usage
